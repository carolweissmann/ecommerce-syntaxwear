# SyntaxWear - Tênis e Sneakers Online

![SyntaxWear Logo](./imagem/logo/logo.svg)

## 🚀 Sobre o Projeto

A **SyntaxWear** é uma interface moderna de e-commerce voltada para o mercado de calçados, especificamente tênis e sneakers. O projeto foca em uma experiência de usuário limpa, responsiva e visualmente impactante, destacando produtos exclusivos e coleções urbanas.

O site foi desenvolvido com foco em performance e organização de código, utilizando uma arquitetura CSS modular para facilitar a manutenção e escalabilidade.

---

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**:
  - **CSS Variables**: Para gestão centralizada de cores e fontes.
  - **Flexbox & CSS Grid**: Para layouts complexos e responsivos.
  - **Arquitetura Modular**: Separação de estilos por componentes (header, footer, hero, etc.).
- **Google Fonts**: Utilização das fontes *Ubuntu* (principal) e *Roboto*.
- **SVG**: Para ícones e logotipos, garantindo nitidez em qualquer resolução.

---

## 📁 Estrutura de Pastas

```text
/
├── css/
│   ├── components/       # Estilos específicos de cada seção (Header, Hero, Footer, etc.)
│   ├── base.css          # Estilos globais e tipografia
│   ├── layout.css        # Estrutura de containers e grids principais
│   ├── reset.css         # Reset de estilos padrões do navegador
│   └── variables.css     # Definição de tokens de design (fontes, cores)
├── imagem/
│   ├── banners/          # Imagens de destaque da Home
│   ├── icon/             # Ícones de redes sociais e utilitários (SVG)
│   ├── logo/             # Identidade visual da marca
│   └── produtos/         # Fotos do catálogo de calçados
└── index.html            # Página principal do projeto
```

---

## ✨ Funcionalidades e Seções

1.  **Cabeçalho Responsivo**: Menu de navegação adaptável com suporte a dispositivos móveis (Menu Hamburguer).
2.  **Seção Hero**: Destaque principal para a linha "Krypton One" com chamadas para ação (CTAs).
3.  **Categorias Dinâmicas**: Acesso rápido a categorias como *Casual*, *Esporte*, *Moderno* e *Futurista*.
4.  **Grid de Produtos**: Exposição de modelos em destaque com layout assimétrico e moderno.
5.  **Rodapé Completo**: Inclui formulário de newsletter, links de redes sociais e navegação estruturada por categorias e informações da loja.

## 🎨 Design System

- **Cores Principais**:
  - Roxo (#6329A2) - Destaque e Interação.
  - Branco (#FFFFFF) - Contrastes e Botões.
  - Cinza Escuro (#333333) - Texto Base.
- **Tipografia**:
  - **Ubuntu**: Utilizada para títulos e corpo de texto, transmitindo modernidade e clareza.
